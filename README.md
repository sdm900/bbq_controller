# BBQController
BBQController is a small project to build a RaspberryPI Zero W to control the pit temperature of a smoker BBQ.

Hardware
* RaspberryPI Zero W
* Pimoroni MCP9600
* Pimoroni 1.12" OLED screen
* Tower Pro micro servo 99 SG90
